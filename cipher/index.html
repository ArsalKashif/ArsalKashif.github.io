ASCII Value Offsets





AOV is a secure encryption method created by ----- ------. It uses a key expansion algorithm to manipulate the key into generating offsets

that modify the original plain text. To encrypt, simply enter in the plain text and the key and click encrypt. Decryption is just as simpl

e, but instead you enter the cipher text. Note that only characters with ASCII values from 32-126 are allowed, otherwise it will not work

as intended.
